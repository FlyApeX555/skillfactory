Основные цели и задачи проекта:
Подготовить данные для создания модели.

Кратка информация о данных:
В данных достаточно чистые, пропусков мало.
Имелись выбросы по 2-м параметрам: absences (количество прогулов), Fedu (образование отца).

Этапы работы над проектом:
1. Первичная обработка (описание функций)
2. Первичный осмотр данных
3. Распределение признака для числовых переменных, устранение выбросов
4. Оценка количества уникальных значений для номинативных переменных, преобразование данных
5. Корреляционный анализ, отбор не коррелирующих переменных
6. Анализ номинативных переменных (на предмет значимости)
7. выводы
